¿Para qué se utiliza la etiqueta abbr?

El elemento abbr representa una abreciación o acronimo.

Diferencias entre los elementos header y h1

El elemento header, sirve para crear la cabecera de la página, en la que se suele incluir el título de la misma. El elemento h1 se utiliza para dar tamaño al texto, es el elemento h que se acompaña de forma jerárquica dentro de la página desde 1 a 6, en orden decreciente de tamaño.
